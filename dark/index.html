the quality gate.

Two Forms of Graph-Level Self-Correction
The mechanism differs.
The safety contract is identical.
ACI: ReAct loop correction
Agent-level retry
ReAct Agent → generate YAML
→ NAC Validate → FAIL
→ agent reads error
→ calls schema template tool
→ regenerates
→ NAC Validate → PASS
SD-WAN: answer-generation retry
Inline node retry
RAG pipeline → generate_answer
→ 3 YAML blocks generated
→ NAC Validate → FAIL (block 3)
→ error injected into next call
→ generate_answer retries
→ NAC Validate → PASS
Both paths run through the same external deterministic validator: NAC Validate. Different correction loops. Same quality gate.
